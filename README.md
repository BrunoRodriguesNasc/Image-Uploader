# IMAGE-UPLOADER

Simple and minimalistic image loader

## Preview 
https://image-uploaderbr.netlify.app/

## Features

- Drag and drop
- Loading 
- image link generator

## Technologies
- React 
- Tailwind css
- Firebase

## Installation

```sh
git clone git@github.com:BrunoRodriguesNasc/Image-Uploader.git
cd imageUploader 
npm install
npm start
```
